Para hacer que funcione el servicio, primero debemos de clonarnos el repositorio a nuestra carpeta local, con un git clone url_del_repo. Una vez tengamos el repo clonado, vamos a powershell y no situamos en la carpeta que hemos clonado (cd ruta_carpeta) y hacemos un docker-compose up para que construya todo el servicio. Una vez este todo creado, en docker desktop iniciamos el contenedor y pulsamos al puerto 8080:80 y se nos mostrara en el navegador la base de datos.
